## Freshco-Retail-Sales-Analysis-Excel

Retail sales data analysis project using Microsoft Excel to identify product performance, seasonal demand trends, and regional sales insights.

## Overview

This project focuses on analysing retail sales data of Freshco Store using Microsoft Excel to generate meaningful business insights. The main objective was to clean and organise raw sales datasets, analyse product category performance, and identify customer purchasing trends across different regions and time periods. By using Excel analytical tools and visualization techniques, the project helps in understanding sales growth patterns and supports effective inventory and marketing decisions.

## Key Features
Cleaned and structured large retail sales datasets to improve data accuracy and usability.
Analysed sales performance across multiple product categories to identify high-performing and low-performing segments.
Used Pivot Tables and Pivot Charts to generate interactive sales summaries and visual reports.
Identified seasonal demand patterns showing increased sales during peak periods.
Performed regional sales analysis to understand market demand variations.
Applied Excel functions to automate calculations and improve reporting efficiency.
